# inactinique.net

Personal academic website of Frédéric Clavert, assistant professor in contemporary history at the [Centre for Contemporary and Digital History (C2DH)](https://c2dh.uni.lu), University of Luxembourg.

Live site: [inactinique.net](https://inactinique.net)

Built with Jekyll and hosted on GitHub Pages. Publications are automatically fetched from [ORBilu](https://orbilu.uni.lu/profile?uid=50029240) via a weekly GitHub Action.
